01

시스템 개요 및 도입 목적

TC 환경관리 시스템은 기존 수기 방식의 문제점(기록 누락, 법적 리스크, 관리 사각지대)을 해결하기 위해 도입된 디지털 환경관리 플랫폼입니다. 현장 데이터를 모바일로 즉시 입력하고 실시간으로 공유합니다.

기존 수기 방식
  • 필터 관리 여부 확인 불가 및 유량 저하 발생
  • 복잡한 보고 단계로 인한 기록 누락 및 법적 처벌(조업정지) 리스크
  • 지적사항 누적 관리 부재 — 동일 항목 반복 지적 및 개선 지연
TC 디지털 시스템
  • 디지털 이력제로 필터 교체 여부 투명성 확보 및 유량 정상 유지
  • 현장 즉시 입력으로 SEMS 기록 누락 방지 및 보고 단계 축소
  • 모바일 사진 기반 히스토리로 현장 개선 여부 상시 확인 가능
02

최초 로그인 설정

처음 시스템을 사용할 때는 관리자에게 부여받은 아이디로 비밀번호를 직접 설정해야 합니다. 이후부터는 설정한 비밀번호로 로그인합니다.

🌐
접속 주소
yjyeom1545.github.io/tc/login.html

🔐 최초 1회 — 비밀번호 설정

1
로그인 화면 하단 First Login 버튼 클릭

로그인 페이지 맨 아래에 있는 First Login 버튼을 누릅니다.

2
부여받은 아이디 입력

관리자에게 사전에 전달받은 아이디를 입력합니다.

3
개인 비밀번호 설정

본인이 사용할 비밀번호를 직접 설정합니다. 설정 완료 후 이 비밀번호를 기억해두세요.

✅ 2회차부터 — 일반 로그인

로그인 화면에서 부여받은 아이디와 직접 설정한 비밀번호를 입력하면 바로 접속됩니다.
※ 비밀번호를 잊어버린 경우 관리자에게 문의하세요.

⚠️ QR 코드는 로그인용이 아닙니다. 현장 설비에 부착된 QR은 필터 교체 기록 폼을 바로 여는 용도입니다. 자세한 내용은 04 섹션을 참고하세요.

04

QR 폼 — 필터 교체 기록 작성법

설비에 부착된 QR을 스캔하면 해당 설비의 여과지(필터) 교체 기록 폼이 바로 열립니다. 폼의 모든 항목은 직접 타이핑이 아닌 선택 방식으로 구성되어 있어 빠르게 입력할 수 있습니다.

탭 A
교체 기록 입력

필터 교체 즉시 작성하는 기본 탭

탭 B
여과지 발주 요청

재고 부족 시 사전 발주 요청 탭

📋 교체 기록 탭 — 입력 순서

1
교체 날짜 선택

실제 필터를 교체한 날짜를 달력에서 선택합니다. 교체 당일 즉시 입력하는 것을 원칙으로 합니다.

2
작업자 이름 선택

본인 이름을 목록에서 선택합니다. 교체를 수행한 담당자가 직접 선택해야 합니다.

3
설비명 선택

교체 작업을 한 설비를 목록에서 선택합니다. 직접 타이핑 없이 드롭다운에서 고르면 됩니다.

4
교체 수량 선택

교체한 여과지의 수량을 목록에서 선택합니다. 해당하는 항목이 없을 경우 관리자에게 문의하세요.

5
저장 버튼 클릭

모든 항목 선택 완료 후 저장 버튼을 눌러 제출합니다. 제출된 데이터는 즉시 DB에 누적되어 관리자에게 실시간으로 공유됩니다.

📦 여과지 발주 요청 탭 사용법

1
여과지 발주 요청 탭으로 전환

QR 폼 상단에서 여과지 발주 요청 탭을 선택합니다.

2
비고란에 필요 수량 입력

요청 비고란에 필요한 수량을 직접 입력합니다.

3
요청 버튼 클릭

요청 버튼을 누르면 관리자에게 발주 요청이 즉시 전달됩니다. 자재 공백으로 인한 교체 지연을 방지하기 위해 재고가 줄어들면 미리 요청하세요.

05

폐유고상 배출 관리

폐유고상이 가득 찼을 때 작업자가 QR을 스캔하여 배출 요청을 올리면, 관리자가 확인 버튼을 눌러 뚜껑을 열어주고 배출 완료까지 처리하는 흐름입니다. 위탁업체 수거 시점도 시스템 내 누적 수량으로 판단합니다.

👷 작업자 — QR 폼 배출 요청

1
폐유고상 가득 참 확인 후 QR 스캔

폐유고상이 가득 찼을 때 설비에 부착된 QR을 스캔합니다. 폐유고상 배출 요청 폼이 바로 열립니다.

2
배출 라인 선택

배출하고자 하는 해당 라인을 목록에서 선택합니다.

3
용기 종류 선택 — 톤백 / 링팩

폐유를 담을 용기 종류(톤백 또는 링팩)와 수량을 선택합니다.

4
배출하기 버튼 클릭

배출하기 버튼을 누르면 사무실 관리자에게 즉시 배출 요청이 전달됩니다. 관리자가 확인할 때까지 잠시 대기합니다.

🖥️ 관리자 — 웹/앱에서 처리

1
배출 요청 알림 확인

작업자가 배출하기를 누르면 관리자 웹/앱에 요청이 표시됩니다. 어느 라인에서 어떤 용기로 몇 개를 배출하려는지 내용을 확인합니다.

2
확인 버튼 클릭 → 뚜껑 개방

웹/앱에서 확인 버튼을 클릭한 뒤 현장으로 가서 폐유고상 뚜껑을 열어줍니다. 작업자가 와서 폐유를 버리거나 관리자가 직접 배출합니다.

3
배출 완료 버튼 클릭

배출이 끝나면 웹/앱에서 배출 완료 버튼을 클릭합니다. 해당 배출 건이 이력에 기록되고 누적 수량에 자동 합산됩니다.

🚛 위탁업체 수거 판단

배출 완료 건이 쌓이면 시스템에서 톤백 / 링팩 총 합계 수량을 자동으로 집계합니다.
관리자는 이 누적 수량을 확인하여 위탁업체를 호출하고 수거를 진행합니다.
※ 수량 집계는 시스템이 자동으로 처리하므로 별도 수기 기록 없이 수거 시점을 정확히 판단할 수 있습니다.
06

대기운영기록 입력 가이드

⚠ 법적 주의사항 운영일지 미작성 및 허위 작성 적발 시 조업정지 10일의 강력한 행정 처분이 부과됩니다. 실시간 입력은 법적 리스크 방지를 위한 필수 의무 사항입니다.

환경 법규 준수 및 SEMS 전송 지연 해소를 위해 매일 아래 항목을 정확히 입력하십시오.

구분 입력 항목 상세 가이드 및 주의사항
운영 정보 운영일자 가동 기록을 작성할 해당 날짜를 정확히 선택
가동 시작/종료 시간 배출구(시설)별 실제 가동 및 종료 시간 입력
운영 상태 정상/비정상 — 시설별 상태에 따라 해당 버튼 클릭
휴무 관리 휴무시간(h) 가동 중단 시 해당 시간 입력 (가동 시간과 대조 필수)
전력 검침 전일검침량 자동 불러오기 적용 (수정 불가 / 읽기 전용)
당일검침량 방지시설 전력계 수치를 확인 후 직접 입력 (필수)

※ 전일 검침 데이터 자동 불러오기 기능을 활용하여 당일 사용량을 정확히 산출하고, 기록 누락으로 인한 법적 불이익이 발생하지 않도록 당일 작성을 원칙으로 합니다.

기대 효과

🛡️
법적 리스크 원천 차단

실시간 데이터 입력으로 SEMS 지연을 해소하고 조업정지 10일 리스크를 사전에 방지합니다.
